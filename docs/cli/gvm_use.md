## gvm use

Switch to a specific Go version

### Synopsis

Switch the current Go environment to the specified version.

Examples:
  gvm use go1.21    # activate Go 1.21
  gvm use           # use version from go.mod (if available)

```
gvm use [version] [flags]
```
